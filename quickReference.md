# Quick Reference

### Key Terms/Concepts

-   **Machine Learning (ML)**: A subset of AI that enables systems to learn from data and improve performance over time without being explicitly programmed.
-   **Neural Networks**: Computational models inspired by the human brain, consisting of interconnected nodes (neurons) that process data.
-   **Supervised Learning**: A type of ML where the model is trained on labeled data, learning to predict outcomes based on input-output pairs.
-   **Unsupervised Learning**: A type of ML that finds patterns in unlabelled data without predefined categories.
-   **Reinforcement Learning**: A type of ML where an agent learns to make decisions by receiving rewards or penalties based on its actions.

### Fundamental Theories

-   **Linear Regression**: A statistical method for modeling the relationship between a dependent variable and one or more independent variables by fitting a linear equation.
-   **Support Vector Machines (SVM)**: A supervised learning model that finds the hyperplane that best separates different classes in the feature space.
-   **K-Nearest Neighbors (kNN)**: A lazy learning algorithm that classifies data points based on the majority class among the k closest training examples.
-   **Convolutional Neural Networks (CNN)**: A class of deep neural networks primarily used for analyzing visual data, employing convolutional layers to extract features.

### Key People

-   **Geoffrey Hinton**: Known as the 'Godfather of Deep Learning', he has made significant contributions to neural networks and deep learning.
-   **Yann LeCun**: A pioneer in convolutional networks and deep learning, known for his work on the MNIST dataset.
-   **Andrew Ng**: Co-founder of Google Brain and a prominent figure in AI and machine learning education.

### Key Algorithms

| Algorithm | Description |
|-----------|-------------|
| Linear Regression | Predicts continuous outcomes by fitting a linear equation to the data. |
| Logistic Regression | Used for binary classification problems, predicting the probability of a class. |
| Decision Trees | A model that makes decisions based on a series of questions about the features of the data. |
| Random Forests | An ensemble method that uses multiple decision trees to improve classification accuracy. |
| Gradient Descent | An optimization algorithm used to minimize the loss function by iteratively adjusting weights. |

### Key Techniques

-   **Cross-Validation**: A technique for assessing how the results of a statistical analysis will generalize to an independent dataset.
-   **Regularization**: Techniques used to prevent overfitting by adding a penalty to the loss function.
-   **Data Augmentation**: A method to artificially increase the size of a training dataset by creating modified versions of existing data.

### Facts to Memorize

-   **Key Machine Learning Algorithms**: Supervised, Unsupervised, Reinforcement
-   **Common Metrics**: Mean Squared Error (MSE), Accuracy Score, F1 Score, Precision, Recall
-   **Activation Functions**: Sigmoid, Tanh, ReLU (Rectified Linear Unit)
-   **Regularization Techniques**: L1 (Lasso), L2 (Ridge)
-   **Common Libraries**: TensorFlow, Keras, Scikit-learn
-   **Hyperparameters**: Learning Rate, Epochs, Batch Size, Number of Neurons, Regularization Parameter

### Reference Information

-   **Machine Learning Types**: Supervised (labelled data), Unsupervised (unlabelled data), Reinforcement (trial and error)
-   **Common Loss Functions**: Cross-Entropy Loss for classification, Mean Squared Error for regression
-   **Common Techniques for Overfitting**: Cross-validation, Regularization, Early Stopping

### Concept Comparisons

| Concept | Description | Example |
|-----------|-----------|---------|
| Supervised Learning | Learning from labelled data with known outputs. | Classifying emails as spam or not based on labelled training data. |
| Unsupervised Learning | Finding patterns in unlabelled data without known outputs. | Clustering customers based on purchasing behavior without prior labels. |
| Reinforcement Learning | Learning through trial and error, receiving rewards or penalties. | Training a game-playing AI that learns to maximize its score through gameplay experience. |
| Eager Learning | Generalizing before seeing the query, creating a model upfront. | Linear regression, where the model is trained before making predictions. |
| Lazy Learning | Waiting for a query before generalizing, storing training data for later. | K-Nearest Neighbors (kNN), which classifies based on the closest training examples. |
